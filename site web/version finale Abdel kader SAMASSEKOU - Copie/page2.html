1) Calculer le somme des chiffre/nombre jusqu'a n

En javascript la fonction for() qui s'écrit for ([initialisation]; [condition]; [expression_finale])Elle prend donc trois parametre s'éparer par des point virgule et qui son tous facultatif(sans aucune condition la boucle devient infini).

function somme_N_entiers (nb){

var somme=0;

for (var i=0; i <=nb ; i+=1){

somme=somme + i ;

}

alert("La somme des chiffre/nombre jusqu'a "+nb+" est "+somme);

}
